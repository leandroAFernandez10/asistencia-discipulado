---- SISTEMA DE ASISTENCIA A CLASES DE DISCIPULADO ----

Este sistema permite gestionar la formación de discípulos dentro de un entorno estructurado por disciplinas, discipulados, clases, matrículas, asistencias y células.
Un discipulado es un curso teórico orientado a una disciplina específica. Cada discipulado cuenta con varias clases. Los discípulos deben matricularse a un discipulado para poder asistir a sus clases. Cada vez que un discípulo asiste a una clase, se registra una asistencia.
Cuando un discípulo alcanza un porcentaje mínimo de asistencias en un discipulado, se habilita su participación en una célula, que es el espacio práctico donde se aplica lo aprendido. Las células también están orientadas a disciplinas, y varios discípulos pueden participar de varias disciplinas y células si cumplen los requisitos.
El sistema permite registrar, modificar y consultar toda esta información para llevar un control completo de la formación de cada discípulo.
